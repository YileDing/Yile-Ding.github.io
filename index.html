📊 进度条功能详解（50%的含义）
进度条是游戏中用于可视化角色能力值的组件，将0-100的数值以图形化方式展示。

实现原理： 
def get_progress_bar(self, value, label, bar_char='#'):
    bar_length = 20                    # 进度条总长度
    percent = value / 100              # 计算百分比
    filled_length = int(bar_length * percent)  # 填充长度
    bar = '#' * filled_length + '-' * (bar_length - filled_length)
    return f"{label}: [{bar}] {value}"

宏观经济数据分析

pandas + matplotlib FRED数据（1948-2026）

基于美联储FRED数据库（CPI、失业率、联邦基金利率），分析美国经济指标，验证菲利普斯曲线。 

import pandas as pd
import matplotlib.pyplot as plt
import numpy as np

# 读取数据
cpi = pd.read_csv('CPIAUCSL.csv', parse_dates=['observation_date'])
unemp = pd.read_csv('UNRATE.csv', parse_dates=['observation_date'])

# 合并并计算通货膨胀率（同比）
df = pd.merge(cpi, unemp, on='observation_date')
df['inflation'] = (df['CPIAUCSL'] / df['CPIAUCSL'].shift(12) - 1) * 100
df = df[df['observation_date'] >= '2000-01-01']

# 绘制菲利普斯曲线散点图 + 趋势线
plt.scatter(df['UNRATE'], df['inflation'], alpha=0.5)
z = np.polyfit(df['UNRATE'], df['inflation'], 1)
p = np.poly1d(z)
plt.plot(x_trend, p(x_trend), 'r--', label=f'Trend (slope={z[0]:.2f})')

print(f'相关系数: {df["UNRATE"].corr(df["inflation"]):.3f}')
📈 关键发现
• 通货膨胀率与失业率呈现负相关（菲利普斯曲线验证）
• 2008年金融危机：联邦基金利率从5.25%降至接近0%
• 2020年COVID-19：失业率飙升至14.8%（历史高位）
• 失业率与通胀率相关系数约 -0.3 ~ -0.5

数列求和：一题多解

7种方法 + 数学公式 时间复杂度分析
方法代码示例时间复杂度适用场景
for循环for i in n: total+=iO(n)通用，易理解
while循环while i<len(n): total+=n[i]O(n)需要索引操作
递归return n[0] + sum_recursion(n[1:])O(n)函数式编程
内置sum()sum(n)O(n)⭐ 最优选择
列表推导式sum([i for i in n])O(n)需要中间列表
生成器sum((i for i in n))O(n)大数据省内存
分治法sum_divide(left) + sum_divide(right)O(n log n)学习并行思想
 
# 等差数列公式（O(1)效率！比循环快无数倍）
def sum_arithmetic(first, last, n):
    return (first + last) * n // 2
# 1+2+...+100 = 5050

# 等比数列公式（O(1)效率！）
def sum_geometric(first, q, n):
    if q == 1: return first * n
    return first * (1 - q ** n) / (1 - q)
# 1+2+4+8+16 = 31.0
💡 算法心得
• 数学公式 > 循环 > 递归（效率对比）
• 内置sum()是Python最推荐的求和方式
• 递归虽然优雅，但存在栈溢出风险
• 分治法适合学习并行计算思想
