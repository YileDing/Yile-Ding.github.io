📈 作品二 · 宏观经济数据分析 (FRED数据)

数据集： CPIAUCSL.csv (CPI) 、 UNRATE.csv (失业率) 、 DFF.csv (联邦基金利率) ，覆盖 1948–2026 年。

  • 📌 通过 pandas.merge 合并 CPI 与失业率，计算同比通货膨胀率
  • 📌 绘制菲利普斯曲线 (散点图 + 趋势线) ，检验通胀与失业负相关
  • 📌 识别 2008 金融危机 (利率骤降) 与 COVID-19 (失业率峰值 14.8%)
核心代码片段
import pandas as pd, numpy as np
cpi = pd.read_csv('CPIAUCSL.csv', parse_dates=['observation_date'])
unemp = pd.read_csv('UNRATE.csv', parse_dates=['observation_date'])
df = pd.merge(cpi, unemp, on='observation_date')
df['inflation'] = (df['CPIAUCSL'] / df['CPIAUCSL'].shift(12) - 1) * 100
df = df.dropna()
corr = df['UNRATE'].corr(df['inflation'])
print(f'失业率与通胀相关系数: {corr:.3f}')   # 输出约 -0.3 ~ -0.5

🔢 作品三 · 数列求和 (7种方法 + 数学公式)
方法代码示意时间复杂度推荐场景
for 循环for i in n: s+=iO(n)通用
while 循环while i<len(n): s+=n[i]O(n)索引操作
递归return n[0]+sum_rec(n[1:])O(n)函数式
内置 sum()sum(n)O(n)✅ 最简洁高效
列表推导式sum([i for i in n])O(n)生成中间列表
生成器sum((i for i in n))O(n)大数据节省内存
分治法sum_divide(left)+sum_divide(right)O(n log n)并行思想
📐 等差数列 / 等比数列 O(1) 公式
等差数列: S = (first + last) * n // 2 → 1+2+...+100 = 5050
等比数列: S = first * (1 - qⁿ) / (1 - q) → 1+2+4+8+16 = 31.0
