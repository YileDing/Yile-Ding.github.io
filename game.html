📖 游戏简介

玩家在24个月（2学年）内，通过每月选择一项活动（学习、健身、社团、社交、兼职、休息），随机触发不同事件，影响五项属性值。游戏包含奖学金、休学、企业单招等特殊机制，并根据最终综合评分给出不同结局。

📊 属性系统（初始值）
📚 学习成绩: 50
💪 身体健康: 50
🧠 心理健康: 50
👥 社会实践: 10
💰 金钱: 500元
📊 进度条功能说明
什么是50%进度条？
进度条将0-100的数值以图形化方式展示，代码实现如下： 
def get_progress_bar(self, value, label, bar_char='#'):
    bar_length = 20
    percent = value / 100
    filled_length = int(bar_length * percent)
    bar = bar_char * filled_length + '-' * (bar_length - filled_length)
    return f"{label}: [{bar}] {value}"

🏆 结局系统
  • 完美结局：final_score ≥ 85
  • 良好结局：final_score ≥ 70
  • 普通结局：final_score ≥ 55
  • 勉强结局：final_score ≥ 40
  • 遗憾结局：final_score < 40
  • 休学结局：health < 20 或 mental < 20
  • 单招结局：social > 80（被大企业提前录取）
